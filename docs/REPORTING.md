# Weekly Landlord Reporting

## Recipient rule

Each building produces its own report and email.

**To**
- landlord asset-management practical owner (manager / assistant manager equivalent)

**Cc**
- landlord asset-management team;
- landlord responsible executive;
- LM team;
- LM responsible executive.

The report is written as an external communication from the LM company to the landlord.

## Report sections

- reporting period;
- key issue;
- changes since last report;
- inquiry/proposal/viewing activity;
- negotiated area/floor changes;
- competitor buildings;
- blocker and pending approval;
- next action, owner, and due date;
- source references;
- attachments or supporting schedule.

## Five investigation commands

1. `통화내용 확인해서 이번주 변동사항 업데이트 해`
2. `이메일 확인해서 이번주 변동사항 업데이트 해`
3. `협의 중인 면적 변동 있는지 확인해`
4. `협의 중인 층 변동 있는지 확인해`
5. `메일이랑 전화 확인해서 경쟁빌딩 파악해봐`

Each command must produce findings, source references, confidence, and a proposed scoped report patch. It must never silently update published operational data.
