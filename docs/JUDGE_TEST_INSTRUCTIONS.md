# Judge Test Instructions

## Demo mode

Use synthetic data only. No company credentials are required.

## Recommended test path

1. Open Admin Web and choose `Data Steward`.
2. Open `Cobalt Finance Center — July update`.
3. Run AI extraction and confirm the four candidate changes.
4. Switch to `Senior Reviewer` and publish.
5. Open Mobile Web and choose `LM Manager`.
6. Import `Broker request — current 5F package`.
7. Confirm request extraction.
8. Verify that plan v1 is blocked and plan v2 is selected.
9. Approve sandbox send.
10. Run the weekly report.
11. Enter `협의 중인 면적 변동 있는지 확인해`.
12. Accept the report patch and approve the external report.

## Reset

Use `Reset demo data` in either surface, or run the documented seed reset command.
